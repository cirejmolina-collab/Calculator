# Calculator App

A simple web-based calculator that performs basic arithmetic operations. The calculator features a clean and responsive design, allowing users to perform calculations directly in their browser.

## Project Description
- Supports addition, subtraction, multiplication, division, and decimal numbers.
- User-friendly interface with clickable buttons for each digit and operation.
- Responsive layout suitable for desktop and mobile browsers.
- Error handling for invalid expressions.

## Instructions to Run the Project
1. **Clone or Download the Project**
   - Place all files (`index.html`, `style.css`, `script.js`) in the Calculator folder.
2. **Run the App**
   - Open `index.html` in your web browser.
   - Use the on-screen buttons to enter numbers and operations.
   - Click `=` to calculate the result, or `C` to clear the display.

## File Structure
- `index.html` — Main HTML file for the calculator UI
- `style.css` — Stylesheet for the calculator's appearance
- `script.js` — JavaScript logic for calculator operations
